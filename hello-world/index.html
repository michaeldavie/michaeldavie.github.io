Setting up a blog

A couple hours ago I thought that I should set up a blog, and it’s already up and running. Fortunately I already had some of the pieces in place (like a Github account and the domain name), but the whole thing was surprisingly simple. Here are the tools that helped:

  • Hugo - Static website generator
  • Clean White Theme for Hugo - Simple blog theme
  • Github Pages - Free hosting from Github
  • Github Actions - Automated workflows triggered on code changes
  • Github Actions for Hugo - Workflow to execute Hugo’s build and update the site
  • Cloudflare DNS - DNS resolution for my domain
  • Let’s Encrypt - Free TLS certificate, managed by Github
